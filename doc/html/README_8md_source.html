38 All test files must be located in the test-files directory and have the prefix
39 "test_" (i.e. test_sample2.txt). The file test_sample2.txt has the following
40 lines in it:
41 
42 > `a = alloc(44K)` <br>
43 > `free(a)`
44 
45 This test case allocates a 64 kilo-byte block of memory and assigns it to the
46 variable 'a'. If the 'K' in the size argument is removed, then this call will
47 only request 44 bytes. This test case then releases the block that is assigned
48 to 'a' with the free command. Variable names can only be one character long,
49 alphabetic letters.
50 
51 Output must match exactly for credit. We have provided some sample output from

53 compare tests against should be located in the test-files directory and must
54 match the name of its corresponding test file with the prefix "result_" instead
55 of "test_". Nothing you add to the code should print to standard output by the
56 time you submit the project.
